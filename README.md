# Photo-Journal

## Chức năng chính
+ **Chụp ảnh** bằng camera thiết bị
+ **Lưu trữ ảnh** cục bộ trên thiết bị
+ **Hiển thị ảnh dạng gallery** kèm tiêu đề.
+ **Xem chi tiết** một ảnh trong bộ sưu tập.
